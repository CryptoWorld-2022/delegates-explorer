FAQ

Voting Rules

One vote per wallet

Only one reserve proof will be valid per wallet address. This means you can only vote for one delegate per wallet address.

Changing your vote

You can change your vote by voting for a new delegate, and the old vote will automatically be replaced with the new vote.

How to vote in the CLI

Once the wallet is loaded, type the following command:

vote DELEGATES_PUBLIC_ADDRESS or DELEGATES_NAME How to register a delegate

Once the wallet is loaded, type the following command:

delegate_register DELEGATE_NAME DELEGATE_IP_ADDRESS How to update a delegates data

Once the wallet is loaded, type the following command:

delegate_update ITEM (about, website, team, shared_delegate_status, delegate_fee and server_specs) Verify Round Statistics

Once the wallet is loaded, type the following command:

verify_round_statistics BLOCK_HEIGHT_OR_BLOCK_HASH
